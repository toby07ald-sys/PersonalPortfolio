Project 1: Embedded Systems & Architecture

Intelligent Traffic Light Controller

Tags: C++, Arduino, Hardware Networking, Mesh Topology

Description: Developed an intelligent traffic light controller using C++ and Arduino. The system utilized hardware networking and mesh topology to optimize traffic flow based on real-time sensor data. In addition, the system incorporates an emergency override mechanism (EVP) that activates when an emergency vehicle strobe pattern is detected. The system is implemented as a single-node unit that can be deployed in a scalable mesh network due to its architecture.

Tags: Python, OOP, Data Management

Description: Developed a comprehensive bus ticketing system using Python and object-oriented programming principles.
